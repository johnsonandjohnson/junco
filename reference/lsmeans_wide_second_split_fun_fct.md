# Second Level Column Split for LS Means Wide Table Layouts

Second Level Column Split for LS Means Wide Table Layouts

## Usage

``` r
lsmeans_wide_second_split_fun_fct(pval_sided, conf_level, include_pval)
```

## Arguments

- conf_level:

  (`proportion`)  
  confidence level of the interval.

- include_pval:

  (`flag`)  
  whether to include the p-value column.
