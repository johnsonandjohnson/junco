# Get Label Map

Maps labels based on the provided label map and split context.

## Usage

``` r
h_get_label_map(.labels, label_map, .var, split_info)
```

## Arguments

- .labels:

  Current labels.

- label_map:

  Mapping for labels.

- .var:

  Variable name.

- split_info:

  Current split information.

## Value

Mapped labels.
