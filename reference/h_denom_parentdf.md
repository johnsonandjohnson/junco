# Get Denominator Parent Data Frame

Retrieves the parent data frame based on denominator.

## Usage

``` r
h_denom_parentdf(.spl_context, denom, denom_by)
```

## Arguments

- .spl_context:

  Current split context.

- denom:

  Denominator type.

- denom_by:

  Denominator grouping variable.

## Value

Parent data frame.
