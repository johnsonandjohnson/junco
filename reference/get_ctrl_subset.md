# Get Control Subset

Retrieves a subset of the DataFrame based on treatment variable and
control group.

## Usage

``` r
get_ctrl_subset(df, trt_var, ctrl_grp)
```

## Arguments

- df:

  (`data.frame`)  
  Data frame to subset.

- trt_var:

  (`character`)  
  Treatment variable name.

- ctrl_grp:

  (`character`)  
  Control group value.

## Value

Subset of the data frame.
