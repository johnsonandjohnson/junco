# Frequency Preparation in Rows

Prepares frequency data in rows based on provided parameters.

## Usage

``` r
h_a_freq_prepinrows(
  x_stats,
  .stats_adj,
  .formats,
  labelstr,
  label_fstr,
  label,
  .indent_mods,
  .labels_n,
  na_str
)
```

## Arguments

- x_stats:

  Statistics data.

- .stats_adj:

  Adjusted statistics.

- .formats:

  Format settings.

- labelstr:

  Label string.

- label_fstr:

  Formatted label string.

- label:

  Label string.

- .indent_mods:

  Indentation settings.

- .labels_n:

  Labels for statistics.

- na_str:

  String for NA values.

## Value

List containing prepared statistics, formats, labels, and indentation.
