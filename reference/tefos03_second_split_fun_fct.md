# Second Level Column Split Function Factory for TEFOS03 (mmy) Table Layout

Second Level Column Split Function Factory for TEFOS03 (mmy) Table
Layout

## Usage

``` r
tefos03_second_split_fun_fct(conf_level)
```

## Arguments

- conf_level:

  (`proportion`)  
  confidence level of the interval.

## Value

Split function to use in the TEFOS03 (mmy) and related table layouts.

## See also

[`tefos03_first_post_fun()`](https://johnsonandjohnson.github.io/junco/reference/tefos03_first_post_fun.md)
for the first level split.
